# Project Context

LearnMate is a local backend-only learning assistant.

Goals:
- upload educational materials: books, articles, documents, video transcripts;
- generate summaries for learning;
- generate quizzes for knowledge retention;
- answer questions over uploaded materials using RAG.

Project purpose:
- help the developer learn LangChain, LangGraph, RAG, and LLM application architecture by building a useful personal app.

Current direction:
- keep the codebase simple and educational;
- prefer clarity over production-level complexity;
- add abstractions only when they solve a real problem.
